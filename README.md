# BOS2.0-project

### 系统结构:
* BOS项目
  - 业务功能模块
    - 基础数据模块
    - 业务受理、调度
    - 分拣管理、流程管理
  - 系统功能模块
    - 用户管理
    - 权限管理
* CRM项目
* 前端系统
---
### 使用技术:
|工具|说明|
|:--:|:--:|
|Struts 2|表现层MVC 框架| 
|Hibernate|数据层持久化框架| 
|Spring、SpringDataJPA|业务管理IoC 和 AOP 框架| 
|Junit|单元测试| 
|jQuery、JQuery Easy UI|JS 框架| 
|Angular|JS 前端 UI 框架| 
|Ztree|JS 树形菜单插件| 
|POI|Office 文档读写组件| 
|CXF|WebService远程调用| 
|ActiveMQ|消息队列| 
|Apache Shiro|权限管理框架| 
|Elasticsearch、SpringDataElasticsearch|搜索服务器| 
|Quartz|任务调度框架| 
|EhCache|缓存技术|
